Steganography :

Greek for 'hidden writing' The art and science of hiding information by embedding messages within other, seemingly harmless messages. Steganography works by replacing bits of useless or unused data in regular computer files such as graphics, sound, text, HTML, or even floppy disks with bits of different, invisible information. This hidden information can be plain text, cipher text, or even images.

 

Revelation :

A user friendly tool, written in Java, that works by altering the least significant bits LSB's of 24-bit bitmap images. As most of the data of a bitmap is encoded in the most significant bits, a few changes in the LSB's has little or no visible effect.

Tips and Tricks:

By default the Java environment supports a maximum of 64MB, essentially this means that Revelation is restricted to the size of images that it can work upon (generally under 5MB). This can be changed by adding:

-Xmx256m (where you want 256 MB's to be the new default memory for Java) to the command line as a parameter.
